# Strategic Decision Lab

A responsive classroom app for teaching data analytics and strategic decision intelligence. Students choose a business scenario, tune decision assumptions, and see the predicted tradeoffs, recommended move, and executive decision brief.

## Classroom Use

- Demonstrate how analytics supports decisions rather than only reports.
- Compare predictive, prescriptive, and strategic reasoning.
- Let student groups defend different assumptions and recommendations.
- Use the generated executive brief for short presentations.

## Scenarios

- Healthcare capacity
- Retail inventory
- Student enrollment
- Fraud response
- Supply chain resilience

## Run Locally

```bash
npm start
```

Open `http://localhost:5173`.

## Test

```bash
npm test
```

## Files

- `index.html` contains the application structure.
- `styles.css` contains the responsive interface.
- `app.js` contains the simulator scenarios, scoring model, and decision brief generation.
